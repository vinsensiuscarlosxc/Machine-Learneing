## Data Exploration and Data Visualization

Data Exploration (Eksplorasi Data) adalah tahap awal dalam analisis data yang bertujuan untuk memahami struktur, karakteristik, dan kualitas data sebelum dilakukan pemodelan. Pada tahap ini dilakukan pemeriksaan terhadap tipe data, jumlah data, missing values, duplikasi, serta distribusi setiap fitur. Selain itu, eksplorasi juga membantu menemukan pola, anomali, dan hubungan antar variabel.

Beberapa aktivitas yang umum dilakukan:

Melihat informasi dataset (shape, info, describe)

Mengecek nilai kosong (missing values)

Mengidentifikasi outlier

Memahami distribusi data

Menganalisis korelasi antar fitur

Data Visualization (Visualisasi Data) adalah proses menyajikan data dalam bentuk grafik atau visual agar lebih mudah dipahami. Visualisasi membantu mengkomunikasikan insight yang ditemukan saat eksplorasi data, serta mempermudah identifikasi pola dan tren.

Contoh visualisasi yang sering digunakan:

Histogram → melihat distribusi data

Boxplot → mendeteksi outlier

Bar chart → membandingkan kategori

Scatter plot → melihat hubungan antar variabel

Heatmap → menunjukkan korelasi antar fitur

## Tujuan Utama

Memahami data secara menyeluruh

Menemukan pola dan insight penting

Mengidentifikasi masalah dalam data

Menjadi dasar untuk preprocessing dan modeling

## Kesimpulan

Data Exploration dan Data Visualization merupakan langkah penting dalam proses analisis data. Tanpa tahap ini, model yang dibangun berisiko kurang akurat karena data belum dipahami dengan baik.
